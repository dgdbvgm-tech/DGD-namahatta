# Цифровая капсула: Нама-хатта

🌐 **Живой сайт:** [https://dgd-namahatta.netlify.app/](https://dgd-namahatta.netlify.app/)

Масштабируемое Single Page Application (SPA) для ведения цифровой летописи регулярных духовных встреч (нама-хатт). Проект создан для сохранения опыта, тем обсуждений и теплой атмосферы, чтобы передавать это будущим поколениям вайшнавов.

## 🚀 Как работает проект (Архитектура)

Проект написан на чистом JavaScript (Vanilla JS), HTML и CSS без использования тяжелых фреймворков. Это обеспечивает максимальную скорость работы и простоту поддержки.

*   **`index.html`** — оболочка приложения с навигацией и поддержкой темной/светлой темы.
*   **`styles/main.css`** — современные стили с использованием эффекта матового стекла (glassmorphism), микро-анимаций и HSL-палитры.
*   **`js/app.js`** — логика приложения. Скрипт загружает данные, генерирует главную страницу (Лента времени) и детальные страницы встреч без перезагрузки браузера (SPA-роутинг по hash).
*   **`data/events.json`** — база данных проекта.

## 📝 Как добавить новую встречу

Вам не нужно писать HTML-код! Чтобы добавить отчет о новой нама-хатте:

1.  Откройте файл `data/events.json`.
2.  Скопируйте структуру любой существующей встречи и вставьте её в начало или конец файла.
3.  Измените данные:
    *   `id`: уникальный идентификатор (например, `"2026-07-15"`).
    *   `date`: дата для отображения (например, `"15 июля 2026"`).
    *   `title`, `subtitle`, `topics` — заполните актуальной информацией.
    *   Добавьте пути к новым фотографиям (например, `photo3.jpg`).
4.  Сохраните файл. Сайт автоматически создаст карточку в ленте и отдельную страницу.

## 💻 Локальный запуск

Для разработки и просмотра сайта на вашем компьютере необходимо использовать локальный сервер (так как браузеры блокируют загрузку локальных JSON-файлов в целях безопасности).

**Способ 1: Использование Python (встроено в Mac)**
Откройте терминал в папке проекта и выполните:
```bash
python3 -m http.server 8080
```
Затем откройте в браузере `http://localhost:8080`.

**Способ 2: Использование VS Code**
Установите расширение **Live Server** и нажмите кнопку "Go Live" в правом нижнем углу окна редактора.
